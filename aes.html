AES is a symmetric key algorithm, which means the same key is used for both encryption and decryption. It supports key sizes of 128, 192, or 256 bits.

Steps in AES Encryption:

  • 1. SubBytes (Byte substitution using S-box)
  • 2. ShiftRows (Shifting rows of the state matrix)
  • 3. MixColumns (Mixing data within columns)
  • 4. AddRoundKey (Combining with encryption key)

These steps are repeated for 10, 12, or 14 rounds depending on the key size.
